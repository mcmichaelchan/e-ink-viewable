E-ink Viewable Settings

Shortcuts

Ctrl+Shift+X — toggle ink style for the current host.
Ctrl+Shift+D — add the current page's main domain (e.g. *.example.com) to whichever list is currently active.
Ctrl+Shift+P — pause / resume ink style globally (overrides white & black lists everywhere).
Customise these on chrome://extensions/shortcuts.

Mode

Choose which list controls where ink style is applied.

Blacklist

Hosts to exclude when in blacklist mode. Use example.com for an exact host, or *.example.com to match all subdomains.

    Whitelist

    Hosts to include when in whitelist mode.
